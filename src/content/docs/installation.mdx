### Requirements

- Astro 4.0 or later
- Tailwind CSS v4
</Prose>

<div class="space-y-4">
<Prose>
### Start fresh

Create a new Astro project with Bearnie pre-configured:
</Prose>

<InstallTabs type="create" command="my-app" />

<Prose>
Then navigate to your project and start developing:
</Prose>

<CodeBlock code={`cd my-app
npm run dev`} lang="bash" />
</div>

<div class="space-y-4">
<Prose>
### Add to existing project

Run the init command to configure your existing Astro project:
</Prose>

<InstallTabs command="init" />

<Prose>
Add the theme styles:
</Prose>

<InstallTabs command="add styles" />

<Prose>
Import the styles in your main CSS file:
</Prose>

<CodeBlock code={`@import "tailwindcss";
@import "./bearnie.css";`} lang="css" />

<Prose>
Then add any component:
</Prose>

<InstallTabs command="add button card dialog" />
</div>

<div class="space-y-4">
<Prose>
### Manual setup

#### 1. Add the cn utility

Create `src/utils/cn.ts`:
</Prose>

<CodeBlock
  code={`export function cn(...classes: (string | undefined | null | false)[]): string {
  return classes.filter(Boolean).join(' ');
}`}
  lang="typescript"
/>
</div>

<div class="space-y-4">
<Prose>
#### 2. Configure path aliases

Add to your `tsconfig.json`:
</Prose>

<CodeBlock
  code={`{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    }
  }
}`}
  lang="json"
/>
</div>

<div class="space-y-4">
<Prose>
#### 3. Add theme styles

Add the styles via CLI:
</Prose>

<InstallTabs command="add styles" />

<Prose>
Then import in your main CSS file:
</Prose>

<CodeBlock code={`@import "tailwindcss";
@import "./bearnie.css";`} lang="css" />
</div>

<div class="space-y-4">
<Prose>
#### 4. Add components

Use the CLI or copy components from `src/components/bearnie/` to your project.
</Prose>

<InstallTabs command="add button card" />

<CodeBlock
  code={`src/
├── components/
│   └── bearnie/
│       ├── button/
│       │   └── Button.astro
│       └── card/
│           ├── Card.astro
│           ├── CardHeader.astro
│           └── CardContent.astro
├── styles/
│   └── bearnie.css
└── utils/
    └── cn.ts`}
  lang="plaintext"
/>
</div>

<div class="space-y-4">

<Prose>
### Updating components

Components and styles are copied to your project — you own them. They don't auto-update when Bearnie releases changes.

To update a component or styles:
</Prose>

<InstallTabs command="add button --yes" />

<Prose>
If you've customized a component, running this will overwrite your changes. Back up your modifications first, then merge manually after updating.

This is intentional. You control when updates happen and what gets changed.
</Prose>
</div>
