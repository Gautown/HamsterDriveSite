仓鼠驱动管家是一个现代化的 Windows 驱动管理工具，采用 Rust 语言开发，提供直观的图形界面，帮助用户轻松管理和维护系统驱动程序。

传统的驱动管理工具通常功能单一、界面陈旧，难以满足现代用户的需求。仓鼠驱动管家采用现代化的技术栈，提供全面的驱动管理功能。

### 主要功能

仓鼠驱动管家提供全面的驱动管理功能，包括：

- **系统信息检测**: 硬件信息、显示器信息、网络适配器等详细信息
- **驱动管理**: 驱动扫描、状态查看、备份恢复、签名验证
- **系统优化**: 依赖分析、问题诊断、性能监控

### 技术特性

采用现代化的技术栈，确保高性能和稳定性：

- **Rust 语言**: 内存安全、高性能的系统级编程语言
- **egui 框架**: 轻量级、跨平台的即时模式GUI
- **Windows API**: 深度集成Windows系统功能

### 用户体验

注重用户体验，提供现代化的界面设计：

- **现代化界面**: 简洁直观的用户界面设计
- **多语言支持**: 完整的中文界面支持
- **高DPI适配**: 支持高分辨率显示器

### 架构设计

采用清晰的架构设计，确保稳定性和可维护性：

- **前后端分离**: 清晰的模块化架构设计
- **异步处理**: 避免UI阻塞，提升响应速度
- **错误处理**: 完善的错误处理和用户反馈
</Prose>

### Theming

Customize colors with CSS variables. Light and dark modes work out of the box:

```css
:root {
  --background: oklch(1 0 0);
  --foreground: oklch(0.145 0 0);
  --primary: oklch(0.205 0 0);
  --primary-foreground: oklch(0.985 0 0);
}

.dark {
  --background: oklch(0.145 0 0);
  --foreground: oklch(0.985 0 0);
}
```
