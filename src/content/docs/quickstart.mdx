# 仓鼠驱动管家快速入门

本指南将帮助您快速上手仓鼠驱动管家，掌握基本操作和核心功能，让您的系统驱动管理更加高效。

## 🚀 首次启动

### 1. 启动程序

- 从桌面快捷方式或开始菜单启动仓鼠驱动管家
- 如果出现用户账户控制(UAC)提示，请点击"是"以管理员身份运行
- 等待程序初始化和加载界面

### 2. 初始设置

- **欢迎界面**: 点击"开始使用"进入设置向导
- **权限确认**: 确认授予程序管理驱动的权限
- **更新设置**: 选择是否启用自动更新检查
- **备份设置**: 配置驱动备份的存储位置

### 3. 首次扫描

- 程序会自动开始首次系统扫描
- 扫描过程中会显示进度条和当前扫描的设备
- 扫描完成后，您将看到系统硬件概览和驱动状态报告

## 📊 主界面导航

### 1. 顶部菜单栏

- **文件**: 包含退出程序、导入/导出设置等选项
- **操作**: 包含扫描、更新、备份等核心操作
- **视图**: 控制界面显示方式和主题
- **设置**: 打开程序设置面板
- **帮助**: 包含帮助文档、关于程序等信息

### 2. 左侧导航栏

- **系统概览**: 显示系统硬件和驱动总体状态
- **驱动管理**: 管理所有已安装的驱动程序
- **硬件信息**: 查看详细的硬件设备信息
- **备份还原**: 管理驱动备份和还原操作
- **系统优化**: 提供系统优化建议
- **设置**: 配置程序参数和行为

### 3. 主内容区域

- 根据选择的导航项显示相应的内容
- 包含详细的信息展示和操作按钮
- 支持表格排序、筛选和搜索功能

### 4. 底部状态栏

- 显示当前程序版本
- 显示系统状态和扫描结果摘要
- 显示当前操作的进度和状态

## 🔧 核心操作

### 1. 驱动扫描

1. **手动扫描**: 点击顶部菜单栏的"操作" > "扫描驱动"
2. **扫描选项**:
   - 快速扫描: 仅检查已安装驱动的状态
   - 完整扫描: 检测所有硬件设备和驱动
   - 自定义扫描: 选择特定类型的设备进行扫描
3. **查看结果**: 扫描完成后，结果会显示在主内容区域

### 2. 驱动更新

1. **检查更新**: 扫描完成后，程序会自动检测需要更新的驱动
2. **查看更新**: 在"驱动管理"页面，需要更新的驱动会标记为黄色
3. **更新驱动**:
   - 单个更新: 点击驱动右侧的"更新"按钮
   - 批量更新: 选择多个驱动，点击"批量更新"按钮
   - 一键更新: 点击顶部的"一键更新所有"按钮
4. **更新过程**: 更新过程中会显示进度，完成后会提示结果

### 3. 驱动备份

1. **选择驱动**: 在"驱动管理"页面选择需要备份的驱动
2. **开始备份**:
   - 单个备份: 点击驱动右侧的"备份"按钮
   - 批量备份: 选择多个驱动，点击"批量备份"按钮
   - 全部备份: 点击顶部的"备份所有驱动"按钮
3. **备份设置**: 选择备份位置和命名方式
4. **查看备份**: 在"备份还原"页面查看所有备份

### 4. 驱动还原

1. **进入备份管理**: 点击左侧导航栏的"备份还原"
2. **选择备份**: 从备份列表中选择需要还原的备份
3. **开始还原**: 点击"还原"按钮，确认还原操作
4. **还原过程**: 还原过程中会显示进度，完成后需要重启系统

### 5. 驱动卸载

1. **选择驱动**: 在"驱动管理"页面选择需要卸载的驱动
2. **开始卸载**: 点击驱动右侧的"卸载"按钮
3. **确认操作**: 确认卸载操作，注意卸载关键驱动可能影响系统
4. **清理残留**: 选择是否清理残留文件和注册表项

## 📈 硬件信息查看

### 1. 系统概览

- 显示CPU、内存、硬盘、显卡等主要硬件信息
- 显示系统版本、架构和激活状态
- 提供硬件性能评估和建议

### 2. 详细硬件信息

1. **处理器信息**: CPU型号、核心数、主频、缓存等
2. **内存信息**: 内存容量、类型、频率、时序等
3. **存储信息**: 硬盘型号、容量、健康状态、分区信息等
4. **显卡信息**: 显卡型号、显存容量、驱动版本、性能参数等
5. **主板信息**: 主板型号、BIOS版本、芯片组等
6. **外设信息**: 键盘、鼠标、打印机等外部设备信息

### 3. 网络设备信息

- 网卡型号和驱动版本
- IP地址和网络连接状态
- 网络适配器的详细参数

### 4. USB设备信息

- 所有连接的USB设备列表
- USB设备的型号、厂商和驱动状态
- USB端口的使用情况

## 📚 常见问题

### 1. 为什么需要管理员权限？

- 管理驱动程序需要修改系统文件和注册表
- 只有管理员权限才能安装和卸载驱动
- 请确保以管理员身份运行程序以使用所有功能

### 2. 驱动更新会影响系统稳定性吗？

- 仓鼠驱动管家会自动备份当前驱动，确保可回滚
- 只推荐经过验证的驱动更新
- 更新前会评估兼容性，降低风险

### 3. 备份的驱动存储在哪里？

- 默认存储在程序安装目录的backups文件夹
- 可以在设置中自定义存储位置
- 建议将备份存储在非系统分区

### 4. 如何处理驱动更新失败？

- 检查网络连接是否正常
- 尝试使用"还原"功能回滚到之前的驱动
- 查看错误信息，根据提示进行故障排除
- 联系技术支持获取帮助

### 5. 程序扫描速度慢怎么办？

- 选择"快速扫描"模式
- 关闭其他占用系统资源的程序
- 定期清理系统垃圾，提高扫描速度
